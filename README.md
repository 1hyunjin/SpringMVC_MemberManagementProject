# SpringMVC_MemberManagementProject

## 회원도메인 요구사항
- 회원을 가입하고 조회할 수 있다.
- 회원은 일반, VIP 2가지 등급이 있다.
- 회원 데이터는 자체 DB를 구축 가능, 외부 시스템과 연동 가능(미확정)

## 주문과 할인 정책 도메인 요구사항
- 회원은 상품 주문 가능
- 회원 등급에 따라 할인 정책 적용
- 모든 VIP는 1000원 할인해주는 고정 금액 할인 적용 (변경 가능성 높음)
- 기본 할인 정책 정하지 못한 상태, 최악의 경우 적용하지 않을 가능성 존재 (미확정)
